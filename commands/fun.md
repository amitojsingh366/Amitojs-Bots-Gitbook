# Fun

## Tictactoe

This command allows you to play Tic Tac Toe with a friend!

_Aliases: 'ttt', 'tictactoe'_

```text
!tictactoe <@username>
!ttt <@username>
```

## Headsortails

This command flips a coin for you!

_Aliases: 'hot', 'headsortails'_

```text
!headsortails
!hot
```

## Dice
